# Sequence Portfolio

A high-end "Scrollytelling" Personal Portfolio Website built with Next.js 14, Framer Motion, and HTML5 Canvas.

## Features

- **High-Performance Canvas Rendering**: Smooth image sequence animations mapped to scroll progress.
- **Premium Aesthetics**: True Black theme with elegant typography and cinematic transitions.
- **Modern Tech Stack**: React 19, Next.js 16, Framer Motion, and Tailwind CSS.
- **Responsive Design**: Fully optimized for all device sizes.
- **Smooth Interaction**: Micro-animations and parallax effects for an immersive experience.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Build

To create a production build:

```bash
npm run build
```
